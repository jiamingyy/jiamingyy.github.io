学习Verilog的基本语法.

可综合语言

可以被综合器转化为相应的逻辑门电路的语句称为可综合语言。

关键字 作用
module/ endmodule Verlog 的基本单元，成对使用
input 输入端口定义
output 输出端口定义
inout 输入/输出端口定义
begin 成对和end使用相当于 {}
end

数据类型

关键字 作用
wire 线网类型
reg 寄存器类型
parameter 参数类型 / 类似于常量

逻辑判断

关键字 作用
if 逻辑判断
else 逻辑判断
case
endcase
default

连续赋值

关键字 作用
always 过程赋值
assign 连续赋值

过程赋值条件

关键字 作用
posedge 上升沿
negedge 下降沿
posedge or negedge 上升沿或者下降沿
